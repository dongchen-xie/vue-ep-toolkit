# ary

::: tip 语法

\_.ary(func, [n=func.length])

:::

## 描述

创建一个调用 func 的函数。调用 func 时最多接受 n 个参数，忽略多出的参数。

## 参数

| 参数 |   类型   |   默认值    |           说明           |
| :--: | :------: | :---------: | :----------------------: |
| func | Function |      -      | 需要被限制参数个数的函数 |
|  n   |  number  | func.length |      限制的参数数量      |

## 返回

- Function: 返回新的覆盖函数

## 例子

```js
_.map(["6", "8", "10"], _.ary(parseInt, 1))
// => [6, 8, 10]
```
