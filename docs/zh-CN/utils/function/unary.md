# wrap

::: tip 语法

\_.unary(func)

:::

## 描述

创建一个最多接受一个参数的函数，忽略多余的参数。

## 参数

| 参数 |   类型   | 默认值 |     说明     |
| :--: | :------: | :----: | :----------: |
| func | Function |   -    | 要处理的函数 |

## 返回

- Function: 返回新的函数

## 例子

```js
_.map(["6", "8", "10"], _.unary(parseInt))
// => [6, 8, 10]
```
