# sample

::: tip 语法

\_.sample(collection)

:::

## 描述

从 collection（集合）中获得一个随机元素。

## 参数

|    参数    |      类型      | 默认值 |     说明     |
| :--------: | :------------: | :----: | :----------: |
| collection | Array / Object |   -    | 要取样的集合 |

## 返回

- any: 返回随机元素

## 例子

```js
_.sample([1, 2, 3, 4])
// => 2
```
