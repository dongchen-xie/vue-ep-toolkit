# takeRight

::: tip 语法

\_.takeRight(array, [n=1])

:::

## 描述

创建一个数组切片，从 array 数组的最后一个元素开始提取 n 个元素

## 参数

| 参数  |  类型  | 默认值 |       说明       |
| :---: | :----: | :----: | :--------------: |
| array | Array  |   -    |   要检索的数组   |
|   n   | number |   1    | 要提取的元素个数 |

## 返回

- array(Array): 返回从末尾开始截取到第 n 个元素的数组切片

## 例子

```js
_.takeRight([1, 2, 3])
// => [3]

_.takeRight([1, 2, 3], 2)
// => [2, 3]

_.takeRight([1, 2, 3], 5)
// => [1, 2, 3]

_.takeRight([1, 2, 3], 0)
// => []
```
