# compact

::: tip 语法

\_.compact(array)

:::

## 描述

创建一个新数组，包含原数组中所有的非假值元素。例如 false, null,0, "", undefined, 和 NaN 都是被认为是“假值”。

## 参数

| 参数  | 类型  | 默认值 |      说明      |
| :---: | :---: | :----: | :------------: |
| array | Array |   -    | 需要处理的数组 |

## 返回

- array (Array): 过滤掉假值的新数组。

## 例子

```js
_.compact([0, 1, false, 2, "", 3])
// => [1, 2, 3]
```
