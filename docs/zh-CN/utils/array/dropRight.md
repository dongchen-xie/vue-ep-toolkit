# dropRight

::: tip 语法

\_.dropRight(array, [n=1])

:::

## 描述

创建一个切片数组，去除 array 末尾的 n 个元素。（n 默认值为 1。）

## 参数

| 参数  |  类型  | 默认值 |      说明      |
| :---: | :----: | :----: | :------------: |
| array | Array  |   -    | 需要处理的数组 |
| [n=1] | number |   1    |  要删除的个数  |

## 返回

- array (Array): 返回`array`的剩余切片

## 例子

```js
_.dropRight([1, 2, 3])
// => [1, 2]

_.dropRight([1, 2, 3], 2)
// => [1]

_.dropRight([1, 2, 3], 5)
// => []

_.dropRight([1, 2, 3], 0)
// => [1, 2, 3]
```
