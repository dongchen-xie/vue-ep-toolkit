# 国际化

Vue EP Toolkit 组件**默认**使用英语。如果你希望使用其他语言，请继续阅读以了解如何操作。

## 全局配置

Element Plus 提供了全局配置选项。

```ts [main.ts]
import VueEpToolkit, { zhCn } from "vue-ep-toolkit"

app.use(VueEpToolkit, {
  locale: zhCn
})
```

## 日期和时间本地化

我们在 `DatePicker` 等组件中使用 [Day.js](https://day.js.org/docs/en/i18n/i18n) 库来管理日期和时间。在 Day.js 中设置正确的语言环境对于国际化正常工作非常重要。您必须单独导入 Day.js 的语言环境配置。

```ts
import "dayjs/locale/zh-cn"
```

[支持的语言列表](https://github.com/element-plus/element-plus/tree/dev/packages/locale/lang)

<ul class="language-list">
  <li>简体中文 (zh-cn)</li>
  <li>美式英语 (en)</li>
</ul>

您只需要在[这里](https://github.com/element-plus/element-plus/tree/dev/packages/locale/lang)添加一个语言文件。
