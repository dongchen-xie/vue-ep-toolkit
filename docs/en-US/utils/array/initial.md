# initial

::: tip Syntax

\_.initial(array)

:::

## Description

获取数组`array`中除了最后一个元素之外的所有元素。

## Parameters

| Parameters  | Type  | Default |      Description      |
| :---: | :---: | :----: | :------------: |
| array | Array |   -    | The array to search |

## Returns

- array(Array): Returns截取后的数组 array

## Examples

```js
_.initial([1, 2, 3])
// => [1, 2]
```
