# tail

::: tip Syntax

\_.tail(array)

:::

## Description

获取除了 array 数组第一个元素以外的全部元素。

## Parameters

| Parameters  | Type  | Default |     Description     |
| :---: | :---: | :----: | :----------: |
| array | Array |   -    | The array to query |

## Returns

- array(Array):Returns除了 array 数组第一个元素以外的全部元素

## Examples

```js
_.tail([1, 2, 3])
// => [2, 3]
```
