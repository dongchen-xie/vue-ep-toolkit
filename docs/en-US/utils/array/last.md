# last

::: tip Syntax

\_.last(array)

:::

## Description

获取 `array` 中的最后一个元素。

## Parameters

| Parameters  | Type  | Default |     Description     |
| :---: | :---: | :----: | :----------: |
| array | Array |   -    | The array to query |

## Returns

- any(any): Returns array 中的最后一个元素

## Examples

```js
_.last([1, 2, 3])
// => 3
```
