# shuffle

::: tip Syntax

\_.shuffle(collection)

:::

## Description

创建一个被打乱值的集合

## Parameters

|    Parameters    |      Type      | Default |     Description     |
| :--------: | :------------: | :----: | :----------: |
| collection | Array / Object |   -    | 要打乱的集合 |

## Returns

- Array: Returns打乱顺序的新数组

## Examples

```js
_.shuffle([1, 2, 3, 4])
// => [4, 1, 3, 2]
```
