# sample

::: tip Syntax

\_.sample(collection)

:::

## Description

从 collection（集合）中获得一个随机元素。

## Parameters

|    Parameters    |      Type      | Default |     Description     |
| :--------: | :------------: | :----: | :----------: |
| collection | Array / Object |   -    | 要取样的集合 |

## Returns

- any: Returns随机元素

## Examples

```js
_.sample([1, 2, 3, 4])
// => 2
```
