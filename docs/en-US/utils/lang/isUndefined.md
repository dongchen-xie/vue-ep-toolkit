# isUndefined

::: tip Syntax

\_.isUndefined(value)

:::

## Description

Checks if `value` 是否是 `undefined`。

## Parameters

| Parameters  | Type | Default |    Description    |
| :---: | :--: | :----: | :--------: |
| value | any  |   -    | The value to check |

## Returns

- boolean: 如果 `value` 是 `undefined`，Returns `true`，否则 `false`。

## Examples

```js
_.isUndefined(void 0)
// => true

_.isUndefined(null)
// => false
```
